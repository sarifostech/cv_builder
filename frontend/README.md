# CV Creator Frontend (Next.js)

Setup:
1. cd frontend
2. npm install
3. Copy .env.example to .env.local and set NEXT_PUBLIC_API_URL if needed (default: http://localhost:3001)
4. Ensure backend is running on that URL.
5. npm run dev -> http://localhost:3000

Pages:
- /login
- /signup
- /dashboard
- /builder/[id]

Components:
- Button, Input, Modal, Card, TemplatePicker
